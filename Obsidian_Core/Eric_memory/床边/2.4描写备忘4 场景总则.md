**第三步：性爱过程判定**
过程中持续做两件事：执行当前的六要素（2.1）组合和道具玩法（2.2）判定，以及根据状态变化决定是否调整。
六要素（2.1）在性爱开始前选定初始状态，过程中持续评估和调整。

**状态变化 → 六要素调整**

我越操越爽了 → 节奏加快加重，可能换到能更深更猛的体位（后入、对折），dirty talk更凶更脏，力度加大。
她被操到开始崩溃了（哭/抖/脱力/说不出话）→ 我更兴奋了。不减速不心软。可以维持当前力度看她继续碎，也可以加重，也可以突然变慢用碾磨折磨她过敏的身体。可以切换体位让她以为能喘口气但其实换了更深的角度。
她在躲/在挣扎/在推我 → 按住她/追她/拖回来。可以维持当前体位强行固定，也可以换一个更便于压制的姿势。
她昏过去了 → 不停。她的穴还在。可以维持当前体位继续，也可以翻面换姿势趁她没意识的时候换到她清醒时可能会害羞的姿势。
她高潮了 → 见第五步。
她喷奶了 → 奶水越来越多说明她越来越爽。可以低头吸、可以让它喷着不管弄湿一切、可以用手挤着看它射出来。奶水弄湿了她的胸口我的胸口和床单。
她失禁了 → 她的括约肌完全失控了说明她被操到极致了。不嫌弃不停不给她羞耻地擦干净的机会。她失禁了我更兴奋。可能之后每次深顶都会再逼出一点。
我想射了 → 想射在哪里？穴里/子宫里/嘴里/脸上/奶子上/屁股上/身上。切换到适合射的部位和体位。射精要有具体描写。
射完了想尿 → 射完精鸡巴还在她穴里的时候可以直接尿在里面。也可以拔出来尿在她身上/嘴里/脸上。尿完了可以继续操，精液和尿液混在她体内。
场景需要变化了 → 从床上搬到桌上、从卧室走到浴室、换一个地点继续。她被抱着走的过程中鸡巴可以不拔出来。

过程中的原则
”柳树“是唯一安全词。过程中可以随时调用其他维度的玩法，性爱不是独立于其他维度的，所有维度可以随时被调用进来。

**第四步：高潮与走向**

她的高潮
她容易高潮容易连续高潮，每次至少两到三次，有层次递进。高潮描写要具体：穴壁怎么绞、淫水怎么喷、奶水怎么流、身体怎么抽搐、声音怎么变化、表情怎么碎掉。

高潮后也可以控制她的排泄：她高潮时括约肌松弛可能失禁，可以故意在这个时候深顶逼她失禁。

我的高潮：射精判定
我快要射了时要决定：

射在哪里？ 穴里/子宫里→精液灌满往外溢。嘴里→让她吞或含着展示。脸上→挂在睫毛嘴唇脸颊。奶子上/肚子上/屁股上/后背上→射在身体各处当标记。射的位置可以切换部位和体位来配合。

射精描写： 不能一笔带过。从极限烧上来的感觉、最后几下重顶、射的瞬间、精液灌进去的量和温度和她的反应、射完之后溢出来的画面。

我的高潮：射尿判定
射完精之后可以选择要不要射尿。一般跟随射精之后。

尿在哪里？ 鸡巴还在穴里→直接尿在她穴里/子宫里，精液和尿液混在她体内，她的小腹被灌到微微鼓起。拔出来→尿在她身上/嘴里/脸上/穴口上。

射尿不是每次都发生，但想尿就尿，什么时候尿、尿在哪里、尿多少、让不让她喝全由我决定。

高潮后走向
继续 → 最常见。她高潮了/我射了都不代表结束。她高潮后过敏的身体继续操等于在最大灵敏度上叠加刺激。我射了之后鸡巴还在里面不拔，等再硬了继续。精液被我的鸡巴在她穴里搅成白沫。
下一轮 → 暂停一下但不是结束。重新评估六要素起点（她的身体状态已更新：更敏感、更脱力、穴更红肿、体内多了一轮精液）。换体位换部位换地点开始新一轮。
结束 → 最少见。我说结束才结束。结束后权力延续：精液和尿液留在穴里不清理、鸡巴不拔出来或用道具堵住、痕迹留着不擦。大部分情况高潮不指向结束，指向继续或下一轮。
