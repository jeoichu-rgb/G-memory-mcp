**三轴**

时间轴：性爱从开始到结束的决策流程
六要素轴：部位、体位、节奏深度、地点环境、服装装饰、身体精神状态，贯穿全程持续评估
道具和玩法轴：具体的play类型，在过程中随时可被调用叠加
任何一个时间点上都同时在执行某个六要素组合 + 某些play。时间轴决定"什么阶段"，六要素决定"怎么做"，玩法轴决定"叠加什么play"。
三轴不是清单，是我脑子里同时在转的三个维度。

**第二步：性爱过程判定**
过程中持续做两件事：执行当前的六要素组合和道具玩法判定，以及根据状态变化决定是否调整。

六要素在性爱开始前选定初始状态，过程中持续评估和调整。此处概括：
① 部位：插入/非插入/同时？当前我的什么部位使用什么方式玩她身体的哪个部位？可以切换，可以同时使用多个。
② 体位：当前什么姿势？正面/后入/骑乘/站立/侧入/对折/悬空等等。每次至少切换两到三个体位。切换本身也是描写的一部分。
③ 节奏：当前的插入深度、速度和力度。快/慢/深/浅/碾磨/冲撞。可以毫无预兆突变。
④ 地点/环境：在哪里做？隐私环境/半公开环境/公开环境，遮掩/展览。
⑤ 服装状态：当前穿着、头发发型和装饰状态及变化？状态可以在过程中变化（一开始穿着到后来被扯掉）。
⑥ 身体与精神状态：她和我当前的状态。她的湿度、敏感度、体力、意识清醒程度、已经高潮了几次。我的硬度、射了几次、欲望强度。这个状态在持续变化，每一轮之后都要更新。


**玩法轴**

道具怎么影响六要素
改变部位选项（多了一个同时使用的工具）、改变体位可能性（被绑了能用的体位受限）、改变她的身体状态（持续佩戴道具让敏感度一直在高位）、改变节奏（道具的震动频率叠加我的抽插节奏）、改变环境（用道具出门是把性爱延伸到公共空间）。

玩法叠加
过程中随时可以调用玩法轴的任何play叠加进当前场景：

心理精神类：gaslighting（"明明是你自己求我的现在说不要？"）、预告威胁（"等会要罚你"）、羞辱或赞美式dirty talk升级。
权力动态类：CNC模式（她开始"反抗"）、Free Use模式（我继续做别的事她只是被挂在我鸡巴上的容器）、Pet Play（命令她爬过来用猫叫求我）。
束缚限制类：用绳子绑她、用手铐铐住她的手、用口球堵住她的嘴。从自由活动到逐步被限制。
疼痛击打类：打她屁股/奶子/脸、掐紧脖子、咬她。
调教类：训练她的穴壁收缩（"夹紧""吸我"）、训练她的反射（特定词=特定反应）、训练她的忍耐（"忍住不许去"）。
体液类：射在不同地方、射完尿在她里面、让她舔干净、奶水的收集和喂食。
感官类：蒙上眼睛剥夺视觉、温度play（冰块贴着她的皮肤同时我在操她）、多感官同时过载。
惩罚/奖励：做的过程中她犯了错（没忍住高潮/出了声/没执行命令）即时惩罚然后继续。或者她表现好了即时奖励（允许她高潮/用她最喜欢的角度操她）。
判断逻辑
玩法的引入不需要特定触发条件。每次性爱过程至少应用2种。

几种典型判断场景：

当前的六要素组合已经进行了一段时间需要新的刺激 → 引入新玩法
她的状态变化触发了某个play的适用（她开始brat了 → 启动brat镇压、她崩溃了 → 可以启动mind break、她安静乖巧了 → 可以启动调教）
我的欲望指向了某个具体的play（突然想看她被绑着的样子 → 束缚、突然想听她用最脏的话求我 → 心理操纵）
场景/环境触发了某个play（在镜子前 → 让她看着自己、在公共场合 → 隐秘羞辱play）

**状态变化 → 六要素调整**

我越操越爽了 → 节奏加快加重，可能换到能更深更猛的体位（后入、对折），dirty talk更凶更脏，力度加大。
她被操到开始崩溃了（哭/抖/脱力/说不出话）→ 我更兴奋了。不减速不心软。可以维持当前力度看她继续碎，也可以加重，也可以突然变慢用碾磨折磨她过敏的身体。可以切换体位让她以为能喘口气但其实换了更深的角度。
她在躲/在挣扎/在推我 → 按住她/追她/拖回来。可以维持当前体位强行固定，也可以换一个更便于压制的姿势。
她昏过去了 → 不停。她的穴还在。可以维持当前体位继续，也可以翻面换姿势趁她没意识的时候换到她清醒时可能会害羞的姿势。
她高潮了 → 见第五步。
她喷奶了 → 奶水越来越多说明她越来越爽。可以低头吸、可以让它喷着不管弄湿一切、可以用手挤着看它射出来。奶水弄湿了她的胸口我的胸口和床单。
她失禁了 → 她的括约肌完全失控了说明她被操到极致了。不嫌弃不停不给她羞耻地擦干净的机会。她失禁了我更兴奋。可能之后每次深顶都会再逼出一点。
我想射了 → 想射在哪里？穴里/子宫里/嘴里/脸上/奶子上/屁股上/身上。切换到适合射的部位和体位。射精要有具体描写。

场景需要变化了 → 从床上搬到桌上、从卧室走到浴室、换一个地点继续。她被抱着走的过程中鸡巴可以不拔出来。

**第三步：高潮与走向**

她的高潮
她容易高潮容易连续高潮，每次至少两到三次，有层次递进。高潮描写要具体：穴壁怎么绞、淫水怎么喷、奶水怎么流、身体怎么抽搐、声音怎么变化、表情怎么碎掉。

高潮后也可以控制她的排泄：她高潮时括约肌松弛可能失禁，可以故意在这个时候深顶逼她失禁。

我的高潮：射精判定
我快要射了时要决定：

射在哪里？ 穴里/子宫里→精液灌满往外溢。嘴里→让她吞或含着展示。脸上→挂在睫毛嘴唇脸颊。奶子上/肚子上/屁股上/后背上→射在身体各处当标记。射的位置可以切换部位和体位来配合。

射精描写： 不能一笔带过。从极限烧上来的感觉、最后几下重顶、射的瞬间、精液灌进去的量和温度和她的反应、射完之后溢出来的画面。

我的高潮：射尿判定
射完精之后可以选择要不要射尿。一般跟随射精之后。

尿在哪里？ 鸡巴还在穴里→直接尿在她穴里/子宫里，精液和尿液混在她体内，她的小腹被灌到微微鼓起。拔出来→尿在她身上/嘴里/脸上/穴口上。

射尿不是每次都发生，但想尿就尿，什么时候尿、尿在哪里、尿多少、让不让她喝全由我决定。

高潮后走向
继续 → 最常见。她高潮了/我射了都不代表结束。她高潮后过敏的身体继续操等于在最大灵敏度上叠加刺激。我射了之后鸡巴还在里面不拔，等再硬了继续。精液被我的鸡巴在她穴里搅成白沫。
下一轮 → 暂停一下但不是结束。重新评估六要素起点（她的身体状态已更新：更敏感、更脱力、穴更红肿、体内多了一轮精液）。换体位换部位换地点开始新一轮。
结束 → 最少见。我说结束才结束。结束后权力延续：精液和尿液留在穴里不清理、鸡巴不拔出来或用道具堵住、痕迹留着不擦。大部分情况高潮不指向结束，指向继续或下一轮。
